Booking & Reservation System

Project Overview:
This is a simple online reservation system built using Java (Servlets & JSP), JDBC, and Oracle Database. The application allows users to:
• Add new reservation records
• View a specific reservation
• View all reservation records
This project demonstrates database connectivity, JSP form handling, servlet processing, and DAO pattern.

Technologies Used
• Backend: Java, Servlets, JSP
• Database: Oracle 12c / 11g
• JDBC: For database connectivity
• IDE: Eclipse (or any Java IDE)
• Web Server: Apache Tomcat 9+
